# `brick-hooks-native`

## Usage

```
const { useAppstate } = require('brick-hooks-native');
```

[useScrollToTop](./src/useScrollToTop/README.md) 方便滚动到顶部

[useStorage](./src/useStorage/README.md) 方便使用react-native的storage
